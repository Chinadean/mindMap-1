CSS Transforms in 2D and 3D
CSS Transforms provide web developers the ability to "transform" elements in 2D and 3D space. Offering more flexibility than just setting the position of an element within its parent, CSS Transforms allow developers to additionally manipulate its rotation, scale and apply skewing distoritions. With the use of the 'perspective' property CSS transforms can give a third dimension to your web pages.

In this section, we will start by introducing the concept of the CSS transform coordinate system. We continue by demonstrating the transform functionality exposed by CSS and wrap-up by illustrating the effect of the 'perspective' property.

Coordinate systems for CSS Transforms

CSS Transforms are defined along a set of axis that form a coordinate system. The x axis of the coordinate system extends along the horizontal dimension of the parent node of the element we are transforming. The y axis extends along the vertical dimension. The point of origin of this coordinate system is originally at the center of the element that we are transforming (the transform-origin property allows developers to change that and we will discuss it later).
