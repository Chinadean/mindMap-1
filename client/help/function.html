mindMap帮助文档

操作和界面说明

  1. 创建新map：file-〉Create New Map 或者使用快捷键[ctrl+m]

  2. 打开硬盘中的map：file-〉Open Map In Disk

  3. 打开storage中的map：file-〉Open Map In Storage

  4. 将map保存到硬盘中：file-〉Save Map In Disk 或者使用快捷键[ctrl+shift+s]

  5. 将map保存到storage中：file-〉Save Map In Storage 或者使用快捷键[ctrl+s]

  6. 选中节点：左键单击节点

  7. nodeContextMenu：右键单击节点

  8. 修改节点位置：左键点击节点不松，拖动鼠标

  9. 修改节点内容：nodeContextMenu-〉edit

  10. 添加子节点：nodeContextmenu-> Append Child Node

  11. 删除节点：nodeContextMenu-〉Delete

  12. 复制节点：选中节点-〉（点击右键-〉Copy）或者（file-〉Copy） 或者使用快捷键[ctrl+c]

  13. 剪切节点：选中节点-〉（点击右键-〉cut）或者（file-〉cut） 或者使用快捷键[ctrl+x]

  14. 粘贴节点：选中节点-〉（点击右键-〉Paste）或者（file-〉Paste） 或者使用快捷键[ctrl+v]

  15. contextMenu: 右键点击map空白处

  16. 取消上一步操作：file-〉Undo 或者 contextMenu-〉Undo 或者使用快捷键[ctrl+z]

  17. 还原上一步操作：file-〉Redo 或者 contextMenu-〉Redo 或者使用快捷键[ctrl+y]

  18. 创建根节点：contextMenu-〉Creat New Root Node
